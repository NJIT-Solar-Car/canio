Detailed Description

Elements values convention: 000000XX000YYYYYb

  • YYYYY : Interrupt flag position in the XX register (5bits)
  • XX : Interrupt status register (2bits)
    • 01: ICSR register
    • 10: SR or SCR or MISR registers
