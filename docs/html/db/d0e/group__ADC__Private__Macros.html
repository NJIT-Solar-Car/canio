◆ ADC_AWD1THRESHOLD_SHIFT_RESOLUTION

#define ADC_AWD1THRESHOLD_SHIFT_RESOLUTION (   __HANDLE__,
  _Threshold_ 
)     ((_Threshold_) << ((((__HANDLE__)->Instance->CFGR1 & ADC_CFGR1_RES) >> 3U)*2U))

Shift the AWD threshold in function of the selected ADC resolution. Thresholds have to be left-aligned on bit 11, the LSB (right bits) are set to 0. If resolution 12 bits, no shift. If resolution 10 bits, shift of 2 ranks on the left. If resolution 8 bits, shift of 4 ranks on the left. If resolution 6 bits, shift of 6 ranks on the left. therefore, shift = (12 - resolution) = 12 - (12- (((RES[1:0]) >> 3)*2))

Parameters
__HANDLE__ADC handle
_Threshold_Value to be shifted
Return values
None
